# Agenda da Naely - Regras de Projeto

- **Identidade Visual**: O design deve ser sempre baseado em Rosé (`#E8C1C5`) e Preto (`#0A0A0A`), mantendo uma estética feminina premium e minimalista.
- **Calendário**: O aplicativo é focado no ano de 2026. Todas as visualizações de data devem respeitar a navegação mensal dentro deste ano.
- **Tecnologia**: React + Vite + Express + Firebase.
- **Notificações**: O backend em `server.ts` contém uma lógica de simulação para o envio de e-mails para `louisenaely@gmail.com`.
- **UX**: Priorizar a facilidade de adicionar tarefas diretamente do calendário.
